Ableton About Page

As part of a front-end development exercise, I recreated the Ableton about page to practice responsive design, HTML5 and CSS3. The project involved building a fully responsive layout using HTML, CSS (Flexbox and Grid), and Bootstrap 5 to match the visual style and user experience of the original site. I focused on replicating key UI components such as the fixed navigation, interactive elements, and seamless scroll behavior, while ensuring performance optimization and accessibility.
