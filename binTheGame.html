Bin The Game Website Showcase

The Bin The Game Showcase website is a responsive site where the main objective is to present the game Bin The Game, with a focus on showing the art of the game. During the development process, the necessary platforms were used to create an interface design that allowed for simple and clear navigation with visual consistency. This resulted in a platform that was light, scalable and easy to maintain.
